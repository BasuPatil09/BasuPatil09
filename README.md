## About

Computer Science Engineering student specializing in **Artificial Intelligence and Machine Learning**. I build end-to-end ML systems — from data ingestion and feature engineering through model training, evaluation, and production deployment via REST APIs and containers.

My engineering standard: modular, config-driven, type-annotated Python. No global state, no hardcoding, no shortcuts. Every design decision is justified by the problem — not by convention.

Currently focused on deepening expertise in model internals, MLOps practices, and algorithmic problem solving in C++17.
